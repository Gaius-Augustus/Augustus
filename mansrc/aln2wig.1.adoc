# aln2wig(1)

## NAME

aln2wig - convert alignments to wig files

## SYNOPSIS

*aln2wig* [parameters] -f filename

## OPTIONS

*-f*::
  Input filename (in psl or shrimp format)

*-s*::
  Use span notation

*-t*::
  Name of the track
